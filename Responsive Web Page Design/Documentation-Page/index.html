Introduction

JavaScript is a versatile programming language that allows you to implement complex features on web pages. It's one of the three core technologies of the web alongside HTML and CSS.

It is used for everything from dynamic content updates to full-scale web applications and server-side development with Node.js.

JavaScript Syntax

JavaScript syntax is the set of rules that define a correctly structured program.

let greeting = "Hello, world!"; if (greeting) { console.log(greeting); }
Variables

Variables are used to store data values. Declared using var, let, or const.

var oldVar = "still works";
let modernVar = "preferred";
const constantVar = "cannot change";
Data Types

JavaScript provides different data types to hold different kinds of values:

  • String
  • Number
  • Boolean
  • Null
  • Undefined
  • Object
  • Symbol (ES6)
let str = "text";
let num = 42;
let isTrue = true;
Operators

Operators are used to perform operations on variables and values.

  • Arithmetic: + - * / %
  • Assignment: = += -=
  • Comparison: == === != !== < >
  • Logical: && || !
let x = 5;
let y = 10;
let result = x + y; // 15
Control Flow

Use control flow to decide what code to execute under specific conditions.

if (score > 90) {
console.log("Excellent!");
} else {
console.log("Keep improving!");
} for (let i = 0; i < 5; i++) {
console.log(i);
}
Functions

Functions let you define reusable blocks of code.

function greet(name) {
return "Hello " + name;
} console.log(greet("Peter"));
Arrays and Objects

Arrays store ordered collections. Objects store key-value pairs.

let fruits = ["apple", "banana", "cherry"];
let person = { name: "Peter", age: 25 }; console.log(fruits[1]); // banana
console.log(person.name); // Peter
DOM Manipulation

JavaScript can access and manipulate the Document Object Model (DOM) to update web pages dynamically.

let heading = document.getElementById("main-title");
heading.textContent = "Updated Title!";
ES6 Features

ES6 (ECMAScript 2015) introduced many helpful features:

  • Arrow functions
  • Template literals
  • Destructuring
  • Classes
  • Promises
const add = (a, b) => a + b;
let name = "Peter";
console.log(`Welcome, ${name}`);
Conclusion

JavaScript is powerful and essential for modern web development. Whether you're building a simple interactive page or a complex application, mastering JavaScript opens up endless possibilities.
